# AB Test Analysis Dashboard

## 📋 Project Description
This project focuses on evaluating the results of an A/B test to determine the impact of changes on key performance indicators. The dashboard provides a comparative framework to analyze user behavior and conversion metrics between different test groups, facilitating data-driven decision-making.

## 🛠 Tools Used
- **Visualization:** Tableau Public
- **Methodology:** Statistical performance comparison, conversion rate analysis, z-statistic and p-value evaluation, and segment breakdown.

## 📈 Key Insights (Academic Project)
* **Statistical Rigor:** Implemented a rigorous evaluation process to distinguish between random fluctuations and meaningful performance impacts by calculating z-statistics and p-values.
* **Comparative Analysis:** Built a comparative framework to track differences in user actions (e.g., `add_payment_info`, `begin_checkout`) across various test iterations.
* **Significant Findings Identification:** Successfully identified statistically significant changes in user conversion rates where p-values indicated reliable outcomes (p < 0.05), providing clarity on which test variations performed effectively.
* **Visualization Proficiency:** Designed an interactive interface for deep-diving into statistical metrics, segmenting performance across devices, channels, and geographies.
